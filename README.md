# 🚀 Courses FastAPI Project

A simple and powerful **FastAPI backend project** for managing courses with full CRUD functionality.

---

## 🌟 Overview

This project is built using **FastAPI** and demonstrates how to create a clean REST API with basic course management features like add, update, delete, and fetch courses.

It is a beginner-friendly backend project suitable for learning API development.

---

## ✨ Features

- ➕ Create new course
- 📄 Get all courses
- 🔍 Get course by ID
- ✏️ Update course details
- ❌ Delete course
- 📦 JSON file-based storage
- ⚡ FastAPI automatic Swagger UI

---

## 🛠️ Tech Stack

- Python 🐍
- FastAPI ⚡
- Uvicorn (Server)
- JSON (Database)
